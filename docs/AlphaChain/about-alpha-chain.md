# About Alpha Chain

Alpha Chain is a dedicated blockchain launched by AlphaToken to facilitate its Real-World Asset (RWA) business. It is an EVM-compatible Ethereum Rollup, built utilizing the modular framework of the OP Stack. This design ensures high compatibility with existing Ethereum tools and infrastructure while offering the scalability and lower transaction costs necessary for efficient RWA tokenization and management.
